# Vuejs-StepByStep-Basic
Vue.js入门和简单的应用    

这个简单的项目是个人Vue.js使用经验的一次总结。一定会有疏漏、偏颇的地方，敬请见谅。当然也希望用到的童鞋可以给些反馈，待我尽力优化，方便后来学习Vue.js的前端同学。

### 目录结构
`/guides` guides目录里是每一次实践的目标和必要的指引    
`/src` src目录是基本的代码
